# 🌤️ Smart Weather Activity Planner

A web app that recommends personalized outdoor activities based on real-time weather conditions at your location.

🔗 **Live Demo:** https://main.d3e49ap3hvje2e.amplifyapp.com/

---

## ✨ Features

- 📍 Detects your location and fetches real-time weather data
- 🤖 AI-powered activity recommendations based on current weather conditions
- 🌧️ Adapts suggestions for rain, sunshine, wind, and more
- 💻 Clean and responsive UI

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS (AI-Assisted) |
| Backend & Hosting | AWS Amplify |
| AI Integration | Generative AI for activity recommendations |

---

## 🚀 How It Works

1. User opens the app and allows location access
2. App fetches real-time weather data for the user's location
3. AI processes the weather conditions and generates personalized activity suggestions
4. User gets a clean list of recommended activities suited for the current weather
